## 📌 Central Idea

After every consultation session, the astrologer (**Dr. Kunwar Harshit Rajveer**) analyses the client's birth chart, numerology, or horoscope and prescribes a personalised set of **remedies**. These remedies can include one or more of the following:

- 🕯️ **Pujas** — sacred rituals to be performed for planetary or spiritual relief
- 📄 **Reports** — detailed written analyses delivered digitally
- 🛎️ **Services** — one-on-one consultation sessions (live or written)

Once the astrologer selects the applicable items, the system **generates a Remedy Report Card** — a structured, printable/shareable receipt that the client receives as their personalised remedial prescription.

## 👤 User & Astrologer Roles — At a Glance

### 🔮 What the Astrologer (Dr. Harshit Rajveer) Does

| Action | Detail |
|--------|--------|
| **Conducts Consultation** | Reads the client's Kundali, numerology, or horoscope during a live session |
| **Diagnoses Doshas / Issues** | Identifies planetary afflictions, karmic patterns, life obstacles |
| **Prescribes Remedies** | Selects pujas, services, or reports as personalised remedies from a catalogue |
| **Adds Personal Notes** | Writes specific instructions, mantra suggestions, lifestyle changes |
| **Generates Report Card** | Confirms selections → system auto-generates a structured receipt/report |
| **Delivers to Client** | Sends the final PDF report card via WhatsApp / email |
| **Tracks via Admin Panel** | Views all consultations, bookings, submissions, and analytics in the admin dashboard |
